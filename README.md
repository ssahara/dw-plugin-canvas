DokuWiki plugin Canvas
======================

Provide HTML5 Canvas functionality for DokuWiki page. Additionally, Inline JavaScript plugin (inlinejs) required to draw something in the canvas.

Syntax
------

    <canvas cid width,height>
    ... javascript ...
    </canvas>
